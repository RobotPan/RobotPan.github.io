# RobotPan: A 360° Surround-View Robotic Vision System for Embodied Perception

**Project Page:** [https://robotpan.github.io](https://robotpan.github.io)

RobotPan is a surround-view robotic vision system that combines six cameras with LiDAR to provide full 360° visual coverage for embodied robot operation. From calibrated sparse multi-view observations, RobotPan predicts metric-scaled and compact 3D Gaussians, enabling real-time surround-view rendering, novel view synthesis, depth estimation, and sparse-view dense reconstruction.

## Key Features

- **Full 360° Coverage** — Six RGB cameras + LiDAR for complete surround-view perception
- **Real-Time Performance** — Feed-forward 3D Gaussian prediction for practical embodied deployment
- **Metric-Scale Reconstruction** — Accurate depth estimation and dense 3D reconstruction
- **Online Fusion** — Long-sequence support with dynamic content updates and bounded static growth
- **Multi-Sensor Dataset** — Covering navigation, manipulation, and locomotion on real platforms
